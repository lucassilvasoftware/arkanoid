# Arkanoid — Unity

Implementação do clássico **Arkanoid** (blocos, raquete e bola) em **Unity**, com cenas de menu, níveis, vitória e derrota. Os scripts em C# controlam a bola, a raquete, os tijolos e o fluxo da partida (`GameManager`, `BallController`, `PaddleController`, etc.).

## Tecnologias

- Unity (versão do editor conforme `ProjectSettings/ProjectVersion.txt`, por exemplo **2023.2.3f1**)
- C#

## Como executar

1. Instale a **Unity Hub** e o editor na mesma linha de versão do projeto.

2. Adicione esta pasta como projeto na Hub e abra-o.

3. Abra a cena de menu principal e a sequência de níveis configurada em **File → Build Settings**.

4. Use **Play** no editor para testar, ou **Build** para gerar o executável na plataforma desejada.
